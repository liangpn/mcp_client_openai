## 项目背景
开发这个适配 OpenAI SDK 的 MCP Client 的原因是，在按照官方示例构建 MCP Client 时，我发现官方示例代码中没有适配 OpenAI SDK 的代码。

---

## 文件说明
- **`client.py`**：适配了 OpenAI SDK 的 MCP Client。
- **`client_new.py`**：为解决我在Windows遇到的问题而适配的版本。
